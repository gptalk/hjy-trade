# hjy-trade

小宇量化 (JZhu Trading) 交易分析平台功能分析

## 应用概述
- **名称**: 小宇量化 - JZhu Trading
- **版本**: v1.0.9
- **技术栈**: React + Spring Boot + TimescaleDB + Docker

## 功能模块
- K线回测 - 股票K线图与技术指标分析
- 策略雷达 - 全市场策略扫描
- 自选股 - 股票分组管理
- 回测记录 - 回测历史查看
- AI问诊 - AI股票分析
- 许愿池 - 用户反馈
- 策略市场 - 策略分享
