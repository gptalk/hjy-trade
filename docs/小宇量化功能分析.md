## 功能模块详解

### 1. K线回测 (首页)
**功能**: 股票K线图展示与技术指标分析

**查询条件**:
- 股票代码输入框 (默认: 000001)
- 市场选择: A股 / 美股 / 港股 / 加密货币 / A股期货
- 周期选择: 1分 / 5分 / 15分 / 30分 / 60分 / 日K / 周K / 月K
- 日期范围: 开始日期 ~ 结束日期

**技术指标按钮**:
- MA (移动平均线)
- EXPMA (指数移动平均线)
- MACD (指数平滑异同移动平均线)
- KDJ (随机指标)

**功能按钮**:
- 查询: 加载K线数据
- 查看A股列表: 查看所有A股
- 策略编辑: 跳转到策略编辑页面
- 策略回测: 执行策略回测
- ⊕ 加入自选股: 添加到自选股列表
- AI策略生成: 使用AI生成策略
- 自动刷新: 自动刷新K线数据

---

### 2. 策略雷达
**功能**: 全市场策略扫描，筛选符合条件的股票

**模式选择**:
- 单策略
- 分批建仓

**策略选择** (预设策略):
- 字母线做多
- 放量突破-做多
- 布尔带突破-做多
- RSI超卖反弹-做多
- MACD金叉死叉-做多
- 均线交叉-做多

**筛选条件**:
- 市场: A股 / 美股 / 港股 / 加密货币
- 周期: 日K / 周K / 月K
- 日期范围: 开始日期 ~ 结束日期
- 杠杆
- 滑点手续费%
- 最少交易次数
- 包含ST/*ST
- 包含新股(<6月)
- 包含指数
- 包含停牌
- 包含北交所

**高级筛选**: 提供更多自定义筛选条件

---

### 3. 自选股
**功能**: 管理用户自选的股票分组

**分组管理**:
- 新建分组 (输入分组名称)
- 分组命名示例: 高股息、半导体核心

**功能**: 
- 将股票添加到指定分组
- 分组查看和管理

---

### 4. 回测记录
**功能**: 查看历史回测结果

**记录字段**:
| 字段 | 说明 |
|------|------|
| 策略 | 使用的策略名称 |
| 代码 | 股票代码 |
| 市场 | A股/美股/港股等 |
| 周期 | 日K/周K/月K |
| 回测区间 | 开始日期 ~ 结束日期 |
| 初始资金 | 回测起始资金 |
| 最终资金 | 回测结束资金 |
| 收益率 | 百分比收益 |
| 胜率 | 盈利交易占比 |
| 交易次数 | 总交易次数 |
| 最大回撤 | 最大亏损幅度 |
| 夏普比 | 风险调整收益 |
| Alpha | 超额收益 |
| 平均持仓 | 平均持仓时间 |
| 最长持仓 | 最长持仓时间 |
| 综合评分 | 综合评级 |
| 运行时间 | 回测执行时间 |

**标签页**:
- 个股回测记录
- 全市场回测记录

**操作**: 删除回测记录

---

### 5. 问诊记录
**功能**: AI股票分析问诊

**计费**: 100积分/次

**使用场景**:
- 分析当前股票走势
- 技术面分析
- 关键支撑/阻力位判断
- 量价关系分析

**分析内容示例**:
- 整体格局判断
- K线形态分析
- MACD/RSI/KDJ指标解读
- 布林带位置
- 关键支撑/阻力位
- 基本面参考 (PE/PB)
- 操作建议

**功能按钮**:
- 复制AI回答 (可直接粘贴到策略编辑)

---

### 6. 许愿池
**功能**: 用户反馈与功能建议系统

**分类**:
- ✨ 功能建议
- 🐛 Bug 反馈
- 📈 策略改进
- ⚡ 性能 / 易用
- 💬 其他

**表单字段**:
- 一句话概括 (0/100字)
- 详细说明 (0/2000字)
- 联系方式 (可选: QQ/微信/邮箱)

**我的提交**: 查看用户提交的历史记录

---

### 7. 策略市场
**功能**: 策略分享与导入市场

**策略卡片信息**:
- 作者 (@用户名)
- 价格 (如: 免费)
- 导入次数
- 策略名称
- 策略描述
- 回测指标:
  - 收益率
  - 胜率
  - 夏普比
  - 最大回撤
  - 交易次数
  - Alpha
- 回测区间

**示例策略**:
1. **RSI超卖-做多** (免费, 已导入12次)
   - 收益率: 29.54%, 胜率: 90.00%, 夏普: 1.03, 最大回撤: 2.83%

2. **黄金坑-做多** (免费, 已导入9次)
   - 收益率: 78.63%, 胜率: 63.64%, 夏普: 1.03, 最大回撤: 11.38%

3. **逆势抢反弹-做多** (免费, 已导入9次)
   - 收益率: 36.12%, 胜率: 72.73%, 夏普: 0.69, 最大回撤: 9.87%

4. **急跌首阳-做多** (免费, 已导入7次)
   - 收益率: 123.53%, 胜率: 83.78%, 夏普: 2.03, 最大回撤: 4.19%

**功能按钮**:
- 浏览市场
- 我的上传
- 一键导入
